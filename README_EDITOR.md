# Markdown编辑器使用说明

## 文件
- `md_editor_qt.py` - Markdown编辑器（支持实时预览和语法高亮）

## 功能特性

### 1. 左右分栏布局
- **左侧**：Markdown编辑器
- **右侧**：实时预览
- **可调节**：拖动中间分隔线调整两边比例

### 2. 实时预览
- 编辑内容会自动在右侧预览
- 使用300ms防拟，避免频繁刷新
- 无需保存文件即可看到效果

### 3. 语法高亮
不同的Markdown语法用不同颜色显示：
- **标题** (`#`) - 蓝色加粗
- **加粗** (`**text**`) - 红色加粗
- **斜体** (`*text*`) - 紫色斜体
- **代码** (`` `code` ``) - 绿色等宽字体
- **代码块** (` ```code``` `) - 青色背景
- **链接** (`[text](url)`) - 蓝色下划线
- **列表** (`- * +`) - 橙色
- **引用** (`>`) - 灰色斜体

### 4. 视图控制
- **隐藏编辑器**：只显示预览（适合演示）
- **隐藏预览**：只显示编辑器（适合专注写作）
- **缩放预览**：Ctrl++ / Ctrl+- / Ctrl+0

## 运行

```bash
python md_editor_qt.py
```

或打开特定文件：
```bash
python md_editor_qt.py README.md
```

## 快捷键

| 功能 | 快捷键 |
|------|--------|
| 新建 | Ctrl+N |
| 打开 | Ctrl+O |
| 保存 | Ctrl+S |
| 另存为 | Ctrl+Shift+S |
| 放大预览 | Ctrl++ |
| 缩小预览 | Ctrl+- |
| 重置缩放 | Ctrl+0 |
| 退出 | Alt+F4 |

## 使用技巧

1. **边写边看**：在左侧编辑时，右侧会实时显示渲染效果
2. **调整布局**：拖动中间分隔线，可以让编辑器或预览占据更多空间
3. **纯编辑模式**：隐藏预览，专注于写作
4. **纯预览模式**：隐藏编辑器，像查看器一样使用
5. **语法提示**：通过颜色高亮快速识别Markdown语法

## 与查看器版本的区别

| 特性 | md_viewer_qt.py | md_editor_qt.py |
|------|-----------------|-----------------|
| 查看Markdown | ✅ | ✅ |
| 编辑Markdown | ❌ | ✅ |
| 语法高亮 | ❌ | ✅ |
| 实时预览 | ❌ | ✅ |
| 分栏布局 | ❌ | ✅ |
| 保存文件 | ❌ | ✅ |

**使用建议**：
- 需要编辑：使用 `md_editor_qt.py`
- 只需查看：使用 `md_viewer_qt.py`
